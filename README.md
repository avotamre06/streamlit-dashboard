# Air Quality Dashboard

This project is an interactive dashboard built with Streamlit to analyze air pollution data from the UCI Air Quality dataset.

## Features

* Interactive pollutant selection (CO, NO2, NOx, C6H6)
* Date range filtering
* Line chart showing pollution trends over time
* Bar chart displaying average pollutant levels
* Clean and user-friendly interface

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy

## Dataset

The dataset used in this project is the UCI Air Quality dataset, which contains real-world measurements of air pollutants collected from sensors.

## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/avotamre06/streamlit-dashboard.git
   ```

2. Navigate to the project folder:

   ```bash
   cd streamlit-dashboard
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the app:

   ```bash
   streamlit run app.py
   ```
